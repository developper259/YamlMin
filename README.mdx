# YamlMin
**YamlMin is a minimized version of yaml.**

You can **write** and **read** yaml files.

supported variable format :
- int 
- float
- string
- bool
- char

Example **code** for **writing** :

	YamlNode node;

	node["str"] = (std::string) "hello world!"; //it is obligatory to cast to std::string else it will return a boolean value 1.
	node["int"] = 1234;
	node["bool"] = true;
	node["char"] = 'h';
	node["float"] = 1.47593f;

	node.write("data.yaml"); //replace 'data.yaml' with the name of your yaml file

**data.yaml** :
	bool: true
	char: h
	float: 1.475930
	int: 1234
	str: hello world!

Exemple **code** for **reading** :

	YamlNode node;

	node.read("data.yaml"); //replace 'data.yaml' with the name of your yaml file

	std::cout << node["int"].getInt() << std::endl;
	std::cout << node["str"].getString() << std::endl;
	std::cout << node["float"].getFloat() << std::endl;
	std::cout << node["bool"].getBool() << std::endl;
	std::cout << node["char"].getChar() << std::endl;

**output** :
	
	1234
	hello world!
	1.47593
	true
	h
